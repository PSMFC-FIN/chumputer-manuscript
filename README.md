This repository includes sample metadata, code, and intermediate output from analyses to reproduce the 
results reported in the manuscript 'Employing deep learning for chum salmon (Oncorhynchus keta) age 
estimation through scale image analysis'. The figures and tables 
in the manuscript can created following the outline in the file ChumputerAnalyses.Rmd within 
the scripts directory. Model training is outlined in Jupyter notebook files (.ipynb),
but was performed with a dedicated high performance computer.
